# Chrome ウェブストア申請用 回答ひな型

- **単一用途の説明**  
  Googleカレンダーの予定タイトル欄に、定型プレフィックスをワンクリック挿入するボタンを追加する。

- **storage が必要な理由**  
  ユーザーが設定したボタンのラベルや挿入テキストを chrome.storage.sync に保存し、端末間で同期するため。

- **ホスト権限（マッチパターン）が必要な理由**  
  Googleカレンダーのタイトル入力欄にボタンを挿入してテキストを入力するために、対象ページの DOM へアクセスする必要がある。対象は `https://calendar.google.com/calendar/*` のみ。

- **リモートコードを使用していますか？**  
  いいえ、リモートコードを使用していません。拡張機能に同梱した JS のみで動作します。

- **データ使用（収集するユーザーデータ）**  
  収集しません。ユーザーが入力したボタン設定をローカルおよび chrome.storage.sync に保存するだけで、個人情報・ブラウジング履歴等は取得しません。

- **3 つの開示への同意**  
  承認されている用途以外で第三者への販売・転送を行いません。唯一の目的以外でユーザーデータを使用・転送しません。信用力判断や融資目的でユーザーデータを使用・転送しません。
